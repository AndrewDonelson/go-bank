## How to run

### Docker

```bash
# Start the database
docker-compose up -d db

# Run the migrations
make migrate-up

# Start the server
make run
```

### Local

```bash
# Start the database
docker-compose up -d db

# Run the migrations
make migrate-up

# Start the server
make run
```

## How to test

```bash
# Run the tests
make test

# Run the tests with coverage
make test-coverage

# Run the tests with coverage and open the browser
make test-coverage-open
```

## How to build

```bash
# Build the binary
make build
```

## How to run the binary

```bash
# Run the binary
./bin/go-bank
```
